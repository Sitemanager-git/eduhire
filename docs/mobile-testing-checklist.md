# Mobile Responsive Testing Checklist

## Screen Sizes to Test[47][50]
- Mobile: 320px - 480px (iPhone SE, Android)
- Tablet: 768px - 1024px (iPad)
- Desktop: 1024px+ (Laptop, Desktop)

## Pages to Test
- [ ] Landing Page
- [ ] Login/Register Forms
- [ ] Teacher Profile Form (all steps)
- [ ] Institution Profile Form (all steps)
- [ ] Job Browse/Search Page
- [ ] Job Detail Page
- [ ] Apply Form Modal
- [ ] My Applications Page
- [ ] Applications Received Page
- [ ] Dashboard (Teacher & Institution)
- [ ] Saved Jobs Page
- [ ] Notifications Panel

## Key Elements to Verify
1. **Navigation**
   - [ ] Hamburger menu on mobile
   - [ ] Menu items accessible
   - [ ] Notification bell responsive

2. **Forms**
   - [ ] Input fields full width on mobile
   - [ ] Buttons easily tappable (min 44px height)
   - [ ] Dropdowns don't overflow
   - [ ] Date pickers mobile-friendly

3. **Tables**
   - [ ] Horizontal scroll if needed
   - [ ] Responsive table alternatives (cards on mobile)

4. **Cards/Lists**
   - [ ] Stack vertically on mobile
   - [ ] Adequate spacing between items

5. **Images**
   - [ ] Scale properly
   - [ ] Don't overflow containers

6. **Charts**
   - [ ] Responsive container
   - [ ] Readable on small screens

## Testing Tools
- Chrome DevTools (Device Toolbar)
- Firefox Responsive Design Mode
- BrowserStack (real devices)
- Physical devices (if available)
