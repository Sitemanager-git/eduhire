# Import Sample Jobs Guide

## Overview
This guide explains how to import sample job data into your MongoDB database.

## Prerequisites
- MongoDB is running
- Node.js is installed
- `.env` file is configured with `MONGO_URI`

## Files Location
- **Sample Data:** `server/data/jobs_sample_data.json`
- **Import Script:** `server/scripts/import-sample-jobs.js`
